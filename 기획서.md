# **📄 \[기획서\] 고양이 성격 유형 밸런스 게임 (CAT-BTI)**

## **1\. 프로젝트 개요**

* **서비스명:** CAT-BTI (Cat Balance Type Indicator)  
* **컨셉:** 10개의 밸런스 질문을 통해 사용자의 숨겨진 '고양이 본능' 유형(4가지)을 찾아내고, 환상의 짝꿍 유형을 매칭해 주는 심리 테스트.  
* **핵심 타겟:** MZ세대, 반려동물 애호가, 커뮤니티 공유를 즐기는 사용자.

---

## **2\. 게임 로직 (Algorithm)**

2개 구간의 선택 결과를 조합하여 4가지 결과 코드를 생성합니다.

| 구간 | 지표 | 알파벳 코드 (A vs B) |
| :---- | :---- | :---- |
| **1구간** | 가치관 | **R**ealistic (현실) vs **I**dealistic (이상) |
| **2구간** | 라이프스타일 | **S**table (안정) vs **A**dventurous (모험) |

*
  **판정 방식:** 각 구간별로 더 많이 선택된 알파벳을 조합 (예: R 3개, A 3개 선택 시 → **RA**)

---

| 구간 | # | 질문 | 왼쪽 선택 | 오른쪽 선택 |
| :---- | :---- | :---- | :---- | :---- |

| | 2 | 100% 확률로 5천만 원 vs 1% 확률로 5억 | **R** | **I** |

| | 4 | 재벌 2세랑 결혼인데 매일 싸움 vs 백수랑 결혼인데 영혼의 단짝 | **R** | **I** |
| | 5 | 배송예정일 미정 맛집 두쫀쿠 주문 vs 후기 0개 두쫀쿠 주문 | **I** | **R** |
| **2구간 (S/A)** | 6 | 평생 내 방에서만 생존 (배달은 됨) vs 평생 밖에서만 생존 (집 출입 금지) | **S** | **A** |
| | 7 | 평생 백반집 한 곳만 vs 매끼 새로운 음식점 (웨이팅 필수) | **S** | **A** |

| | 10 | 태어난 동네에서 무덤까지 vs 1년마다 다른 나라로 강제 이사 | **S** | **A** |

## **4\. 4가지 결과 유형 및 짝꿍 매칭**

| 코드 | 유형 명칭 | 짝꿍 코드 (2구간 반전) |
| :---- | :---- | :---- |
| **RS** | 서울 자가 대기업 김부장냥이 | **RA** |
| **RA** | 레이타고 전국일주 모험냥이 | **RS** |
| **IS** | 로또사고 집콕 몽상냥이 | **IA** |
| **IA** | 인생은 한방 타짜냥이 | **IS** |

## **5\. 유형별 상세 설명**

### **RS - 서울 자가 대기업 김부장냥이**
- 5년 후 벤츠? ㅋㅋ 지금 당장 레이 뽑음
- 라면 물 550ml 계량컵 필수. 눈대중? 그런 거 없음
- 10년째 같은 백반집 단골. 사장님이 메뉴 안 물어봄
- 흑역사 삭제 >>> 로또 번호 0.05초 (어차피 못 외움)
- 태어난 동네에서 영면할 예정. 이사는 귀찮아.

**짝꿍 (RA):** 둘 다 현실적이라 대화가 통함. 김부장이 계획 짜면 모험냥이가 실행함. 완벽한 분업 👍

### **RA - 레이타고 전국일주 모험냥이**
- 일단 레이 뽑고, 전국 맛집 투어 시작
- 매달 새로운 취미 입문. 집에 장비만 산더미
- 웨이팅 3시간? 새로운 맛집이면 참을 수 있음
- 1년마다 다른 동네로 이사. 택배기사님이 날 모름
- 확실한 5천만 원 받고, 모험은 내 발로 직접 함

**짝꿍 (RS):** 모험냥이가 신나게 돌아다니는 동안 김부장이 집 지킴. 돌아오면 따뜻한 밥이 있음 🏠

### **IS - 로또사고 집콕 몽상냥이**
- 5년 후 벤츠 기다리는 중. 상상만 해도 행복함
- 로또 번호 0.05초면 충분해 (뇌에 새길 자신 있음)
- 방에서 나가기 싫음. 배달이 있잖아
- 10년째 레고 조립 중. 아직 완성 안 함
- 백수랑 영혼의 단짝 > 재벌이랑 매일 싸움

**짝꿍 (IA):** 둘 다 로또 당첨 꿈꾸는 중. 타짜냥이가 복권 사오면 몽상냥이가 번호 확인함. 꿈은 같이 꾸는 거야 🎰

### **IA - 인생은 한방 타짜냥이**
- 1% 확률 5억? 당연히 도전. 인생은 한방임
- 5년 후 벤츠 타고 세계일주 할 예정 (예정)
- 맛집 두쫀쿠 배송 미정이어도 기다림. 맛이 중요하지
- 매달 새 취미 + 1년마다 새 나라. 정착은 나중에
- 로또 번호 0.05초? 난 외울 수 있음 ㅇㅇ

**짝꿍 (IS):** 타짜냥이가 세계일주하고 오면 몽상냥이가 집에서 기다림. 모험담 들으며 같이 꿈꿈 ✈️
