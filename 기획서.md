# **📄 \[기획서\] 고양이 성격 유형 밸런스 게임 (CAT-BTI)**

## **1\. 프로젝트 개요**

* **서비스명:** CAT-BTI (Cat Balance Type Indicator)  
* **컨셉:** 20개의 밸런스 질문을 통해 사용자의 숨겨진 '고양이 본능' 유형(16가지)을 찾아내고, 환상의 짝꿍 유형을 매칭해 주는 심리 테스트.  
* **핵심 타겟:** MZ세대, 반려동물 애호가, 커뮤니티 공유를 즐기는 사용자.

---

## **2\. 게임 로직 (Algorithm)**

4개 구간의 선택 결과를 조합하여 16가지 결과 코드를 생성합니다.

| 구간 | 지표 | 알파벳 코드 (A vs B) |
| :---- | :---- | :---- |
| **1구간** | 가치관 | **R**ealistic (현실) vs **I**dealistic (이상) |
| **2구간** | 라이프스타일 | **S**table (안정) vs **A**dventurous (모험) |
| **3구간** | 관계 기술 | **T**hinking (이성/효율) vs **F**eeling (감성) |
| **4구간** | 사회적 자아 | **P**rivate (개인) vs **C**ommunity (공동체) |

*   
  **판정 방식:** 각 구간별 5문항 중 더 많이 선택된 알파벳을 조합 (예: R 3개, A 4개, T 5개, P 3개 선택 시 → **RATP**)

## **3\. 밸런스 게임 문항 (20개)**

| 구간 | 질문 (A vs B) |
| :---- | :---- |
| **1구간 (R/I)** | 1\. 연봉 1억 평범녀/남 vs 연봉 3천 하고 싶은 일 하기 2\. 100% 확률 1천만 원 vs 1% 확률 100억 3\. 과거 지우기 능력 vs 5초 미래 보기 능력 4\. 돈 많은 결혼 vs 사랑 넘치는 결혼 5\. 외모 평범 천재 vs 잘생긴/예쁜 평범인 |
| **2구간 (S/A)** | 6\. 평생 집콕(배달 가능) vs 평생 노숙 세계여행 7\. 단골 맛집 가기 vs 3시간 웨이팅 신상 맛집 8\. 무인도 혼자 생존 vs 좀비 사태 10명과 생존 9\. 매뉴얼대로 일하기 vs 새로운 방식 도전하기 10\. 평생 여름 나라 vs 평생 겨울 나라 |
| **3구간 (T/F)** | 11\. "고생했어" 위로받기 vs "해결책 줄게" 조언받기 12\. 우는 친구랑 같이 울기 vs 맛있는 거 사주기 13\. 내가 읽씹하기 vs 상대가 읽씹하기 (더 나은 쪽) 14\. 실용적인 가전 선물 vs 로맨틱한 꽃다발/편지 15\. 즉시 대화로 풀기 vs 시간을 갖고 나중에 풀기 |
| **4구간 (P/C)** | 16\. 나 홀로 성과 내고 독식 vs 다 같이 협업하고 공유 17\. 내가 먹고 싶은 메뉴 vs 다수가 원하는 메뉴 18\. 분 단위 계획 여행 vs 무계획 즉흥 여행 19\. 1시간 늦는 친구 vs 장소 맘대로 바꾸는 친구 (더 분노) 20\. 찐친 1명만 두기 vs 얕은 인맥 1,000명 두기 |
