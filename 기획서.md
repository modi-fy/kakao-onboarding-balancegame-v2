# **📄 \[기획서\] 고양이 성격 유형 밸런스 게임 (CAT-BTI)**

## **1\. 프로젝트 개요**

* **서비스명:** CAT-BTI (Cat Balance Type Indicator)  
* **컨셉:** 10개의 밸런스 질문을 통해 사용자의 숨겨진 '고양이 본능' 유형(4가지)을 찾아내고, 환상의 짝꿍 유형을 매칭해 주는 심리 테스트.  
* **핵심 타겟:** MZ세대, 반려동물 애호가, 커뮤니티 공유를 즐기는 사용자.

---

## **2\. 게임 로직 (Algorithm)**

2개 구간의 선택 결과를 조합하여 4가지 결과 코드를 생성합니다.

| 구간 | 지표 | 알파벳 코드 (A vs B) |
| :---- | :---- | :---- |
| **1구간** | 가치관 | **R**ealistic (현실) vs **I**dealistic (이상) |
| **2구간** | 라이프스타일 | **S**table (안정) vs **A**dventurous (모험) |

*
  **판정 방식:** 각 구간별로 더 많이 선택된 알파벳을 조합 (예: R 3개, A 3개 선택 시 → **RA**)

---

## **3\. 밸런스 게임 문항 (10개)**

| 구간 | # | 질문 | 왼쪽 선택 | 오른쪽 선택 |
| :---- | :---- | :---- | :---- | :---- |

| | 2 | 100% 확률로 5천만 원 vs 1% 확률로 5억 | **R** | **I** |

## **4\. 4가지 결과 유형 및 짝꿍 매칭**

| 코드 | 유형 명칭 | 짝꿍 코드 (2구간 반전) |
| :---- | :---- | :---- |
| **RS** | 서울 자가 대기업 김부장냥이 | **RA** |
| **RA** | 레이타고 전국일주 모험냥이 | **RS** |
| **IS** | 로또사고 집콕 몽상냥이 | **IA** |
