# **📄 \[기획서\] 고양이 성격 유형 밸런스 게임 (CAT-BTI)**

## **1\. 프로젝트 개요**

* **서비스명:** CAT-BTI (Cat Balance Type Indicator)  
* **컨셉:** 20개의 밸런스 질문을 통해 사용자의 숨겨진 '고양이 본능' 유형(16가지)을 찾아내고, 환상의 짝꿍 유형을 매칭해 주는 심리 테스트.  
* **핵심 타겟:** MZ세대, 반려동물 애호가, 커뮤니티 공유를 즐기는 사용자.

---

## **2\. 게임 로직 (Algorithm)**

4개 구간의 선택 결과를 조합하여 16가지 결과 코드를 생성합니다.

| 구간 | 지표 | 알파벳 코드 (A vs B) |
| :---- | :---- | :---- |
| **1구간** | 가치관 | **R**ealistic (현실) vs **I**dealistic (이상) |
| **2구간** | 라이프스타일 | **S**table (안정) vs **A**dventurous (모험) |
| **3구간** | 관계 기술 | **T**hinking (이성/효율) vs **F**eeling (감성) |
| **4구간** | 사회적 자아 | **P**rivate (개인) vs **C**ommunity (공동체) |

*   
  **판정 방식:** 각 구간별 5문항 중 더 많이 선택된 알파벳을 조합 (예: R 3개, A 4개, T 5개, P 3개 선택 시 → **RATP**)
